# Human Interface Guidelines for The Data City

A set of practices and standards for designing The Data City products for local and international markets.

This repository contains the Human Interface Guidelines (HIG) as Markdown files (in `docs/`), and
is powered by Jekyll to generate a static documentation site.


## Edit locally

Make sure Ruby is installed on your machine. Then run:

    bundle install

To serve locally:

    bundle exec jekyll serve

Then visit `http://localhost:4000` in your browser.
